# SourceMod: Admin Logging

A fork of [Bailopan's plugin](https://forums.alliedmods.net/showthread.php?t=60471).

A plugin that logs all admin commands to a file with the admnin's Steam ID.

## Usage

### Development

- The `LogAction` function must be used as the logging method.

## Installation

Follow the standard SourceMod process for installation by adding the compiled plugin `admin_logging.smx` to `tf/addons/sourcemod/plugins/` and reloading all plugins or restarting the server.

### Dependencies

- SourceMod v1385 or later.
